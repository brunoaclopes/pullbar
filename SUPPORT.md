# Support

## Getting Help

- Open a GitHub issue for bugs or feature requests.
- Include environment details (macOS version, app version/tag, install method).

## Before Opening an Issue

- Run `swift build`
- Validate scripts:
  - `./scripts/run-signed.sh --no-run`
  - `./scripts/install.sh --help`
- Check project docs:
  - `docs/ARCHITECTURE.md`
  - `docs/OPERATIONS.md`
